# ♻️ ZeroWaste Connect

**ZeroWaste Connect** is a platform that bridges the gap between restaurants with surplus food and NGOs that distribute it to those in need. The system facilitates food donations in a seamless, trackable, and efficient manner.

---

## 🌟 Features

- 🍱 **Restaurant Dashboard**: Add leftover food, available packets, and track NGO orders.
- 🏥 **NGO Dashboard**: View available food items and place pickup requests.
- 🔒 **JWT Authentication**: Secure login and access control for all user roles.

---

## 🚀 Tech Stack

| Frontend | Backend | Database | Auth |
|----------|---------|----------|------|
| React + Tailwind CSS | Django REST Framework | PostgreSQL | JWT |

---





## 🌍 How It Works

- ✅ Register as a restaurant or NGO  
- 🍲 Restaurants add details of leftover food  
- 📦 NGOs browse available food and place pickup requests  
- 📉 Inventory updates automatically after pickup confirmation
